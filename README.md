Password Auditor in Python

Overview 
This Python project evaluates the strength of a password by assigning it a score from 0 to 10. It also labels the password strength as weak, moderate, strong, or very strong.

Features
- Checks password length
- Detects uppercase length
- Detects lowercase letters
- Detects numbers
- Detects special characters
- Detects common passwords
- Detects repeated characters
- Provides improvement recommendation

Technology used 
- Python
- Regular Expressions 

How it works 
The program analyzes a password based on common security rules. It gives points for stronger password characteristics and provides feedback on how to improve weaker passwords.
